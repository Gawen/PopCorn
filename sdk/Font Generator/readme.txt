Auteur: Antoine
Type: programme

description: un convertisseur de polices windows en fichiers PCF (PopCornFont et pas autre chose)
Il permet un lissage très important des formes, meme pour les polices non true type.
Il exporte dans un fichier bitmap multiple qui n'enregistre que la couche alpha, et qui permet de ne pas enregistrer tous les caractères.
